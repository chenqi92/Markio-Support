常见问题

我的笔记保存在哪里？
保存在你自己挂载到 Markio 的目录里，是普通的 .md 文件，可以用任何编辑器直接打开。

API Key 安全吗？
Markio 把 API Key 写入 macOS 钥匙串，不会出现在配置文件、localStorage 或日志中。

不联网能用吗？
可以。只要不配置 AI 或同步服务，Markio 完全离线工作。

FAQ

Where are my notes stored?
In the folders you mount into Markio, as plain .md files you can open with any editor.

Are API keys safe?
Markio stores API keys in the macOS keychain. They never land in config files, localStorage, or logs.

Can I use it offline?
Yes. Without configuring AI or sync providers, Markio runs entirely offline.
